## Practical Fit

TypeScript is strongest when JavaScript reach is required but a team needs earlier feedback for API shape, refactors, and cross-file contracts. The type system is erased during compilation, so external input still needs runtime validation.

## Tooling

The TypeScript compiler checks types and emits JavaScript. Most projects use it through npm-compatible package managers and bundlers.

```ts
type Language = {
  name: string;
  runtime: string;
};

const language: Language = {
  name: "TypeScript",
  runtime: "JavaScript",
};

console.log(`${language.name} targets ${language.runtime}`);
```
